# Habit Tracker

A personal habit tracking Progressive Web App for managing recurring tasks with flexible scheduling.

## Features

- **Flexible frequencies** — daily, weekly, X times per week, biweekly, monthly, X times per month, or every N days
- **Context filtering** — tag tasks as work, personal, or both, and filter your view accordingly
- **Completion stats** — weekly and monthly progress with visual completion rings
- **Mobile-first PWA** — installable on iOS and Android, works offline-capable
- **Undo** — quickly reverse accidental completions

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS v4
- Vite (build + dev server)
- Supabase (PostgreSQL database + email OTP authentication)
- Deployed to GitHub Pages

## Status

This is a personal project and is not intended for general use. It requires a configured Supabase backend with specific tables and row-level security policies.
